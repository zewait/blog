1.2 持久层: mongodb

1.2.1 优势

  • 快速!(当然，这和具体的应用方式有关，通常来说，它比一般的关系型数据库快5位左右。)很高的可扩展性 – 轻轻松松就可实现PB级的存储(但是可能我们并不需要PB级的存储，10TB可能就够了)
  • 它有一个很好的 replication 模式 (replica sets)
  • 有很完善的Java API
  • 它的存储格式是Json的，这对Java来说非常好处理，对javascirpt亦然。
  • 它有一个非常活跃的社区
  • 扩展性高

1.2.2 Getting Started Guide

  1. mongodb website: https://www.mongodb.org/
  2. 安装: https://docs.mongodb.org/manual/installation/
  3. CRUD概念: https://docs.mongodb.org/manual/core/crud/
  4. 传统关系型数据库跟mongodb的使用对比:https://docs.mongodb.org/manual/reference/sql-comparison/
  5. java版本操作库: https://docs.mongodb.org/getting-started/java/

2.3 记录数据

* 数据范例(取于nginx 访问日志)

  • 格式 '$remote_addr - $remote_user [$time_local] "$request" $status $body_bytes_sent "$http_referer" "$http_user_agent"
  • 例子 127.0.0.1 - - [05/Oct/2015:00:00:58 +0800] "GET /index.php HTTP/1.1" 200 91632 "-" "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_10_2) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/45.0.2454.101 Safari/537.36"

说明:

  • $remote_addr - client address
  • $remote_user - user name supplied with the Basic authentication
  • $time_local - local time in the Common Log Format (1.3.12, 1.2.7)
  • $request - full original request line
  • $status - response status (1.3.2, 1.2.2)
  • $body_bytes_sent - number of bytes sent to a client, not counting the response header; this variable is compatible with the “%B”
  • $http_referer - http header "Referer"
  • $http_user_agent - http header "User-Agent"
